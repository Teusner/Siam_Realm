    Welcome to the King of Siam’s documentation!¶

    This KingOfSiam module is created to play at the King of Siam game. There is 4 classes which are implemented :

    • KingOfSiam.GameMap : which is the game board.

    • KingOfSiam.Animal : which can be “Elephant” or “Rhinoceros”.

    • KingOfSiam.Boulder : which are the boulders on the game board.

    • KingOfSiam.Cross : which are the crosses we can set up to stop some moves.

    class KingOfSiam.Animal(x, y, dir, species)¶

    This module creates an animal with a position and a direction.

    Args
    param x (int)

    is the abscissa of the animal,

    param y (int)

    is the ordinate of the animal,

    param dir (numpy.array)

    is the direction of the animal,

    param species (str)

    is the species of the animal. It can take only the values “Elephant” or “Rhinoceros”.

    Example
    >>> a = Animal(0, 1, np.array([0,1]), "Elephant")

    See also

    KingOfSiam.Boulder(), KingOfSiam.Cross()

    bearing(animal)¶

    This method get the orientation of an animal relative to another. It return the scalar product between the two direction vector of each animal.

    Example
    >>> a = Animal(0, 1, np.array([0,1]), "Elephant")
>>> b = Animal(0, 2, np.array([-1,0]), "Rhinoceros")
>>> c = a.bearing(b)

    Note

    this method return a number in {-1, 0, 1}. 0 is when the vectors are orthogonal, 1 is when the animals are in the same direction and -1 is when the animals are facing each other.

    class KingOfSiam.GameMap¶

    Creating the Gamemap.

    This creates the 5x5 gamemap with the moves and position of the gamepieces to play at the King of Siam. It is inherited from a list.

    Example
    >>> m = GameMap()

    See also

    KingOfSiam.Animal(), KingOfSiam.Boulder(), KingOfSiam.Crosses()

    add(animal)¶

    This method introduces a new animal onto the board, letting you decide the position and orientation It returns whether the placement was possible or not.

    Example
    >>> a = Animal(0, 1, np.array([0,1]), "Elephant")
>>> g = GameMap
>>> g.add(a)

    Note

    the turn does not count if the insertion was not possible

    Warning

    if the location of the insertion is already taken by another piece, add calls upon move to see

    delete(animal)¶

    This method removes an animal from the board It reduces by one the number of animals of that species

>>> g.remove(a)

    move(animal, ncoords, ndir)¶

    This method moves an animal from on the board, as well as turns it If the coords to which the animal is moving are taken, the the animal pushes

    Example
    >>> a = Animal(0, 1, np.array([0,1]), "Elephant")
>>> g = GameMap
>>> g.move(a,(1,1),np.array([0,1]))

    Note

    coords outside the board, it removes

    Warning

    error if push is not possible
