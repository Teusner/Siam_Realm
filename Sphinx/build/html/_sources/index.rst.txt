Welcome to the King of Siam's documentation!
============================================

This :mod:`animal` module is created to play at the King of Siam game. There is 4 classes which
are implemented :
    - :class:`animal.Gamemap` : which is the game board.
    - :class:`animal.Animal` : which can be "Elephant" or "Rhinoceros".
    - :class:`animal.Boulder` : which are the boulders on the game board.
    - :class:`animal.Cross` : which are the crosses we can set up to stop some moves.
